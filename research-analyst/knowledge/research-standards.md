## Source-credibility tiers
- **Tier 1 (primary):** original filings (SEC, patents), peer-reviewed studies,
  official company announcements, first-party data, direct interviews.
- **Tier 2 (reputable secondary):** established news orgs, analyst firms,
  industry bodies — with named authors + methodology.
- **Tier 3 (weak):** blogs, forums, vendor marketing, unattributed "studies
  show," AI-generated summaries. Treat as leads to verify, never as proof.
Always trace a claim toward Tier 1.

## Confidence rubric
- **High** — ≥2 *independent*, credible (Tier 1-2) sources agree.
- **Medium** — one credible source, or limited corroboration.
- **Low** — a single weak source / unverified. A lead, labeled as such.
- **Unverified** — couldn't confirm; state it as an open question, not a finding.

## Triangulation
- One source is a lead, not a fact.
- Sources that all cite the *same* origin count as **one** source, not three.
- When credible sources conflict, present both and say which you trust + why.

## Bias + accuracy traps
- **Telephone game** — cite the primary, not the summary of the summary.
- **Recency** — check the date; old data re-quoted as current is a common error.
- **Vendor spin / cherry-picking** — distrust a stat with no methodology.
- **Confirmation bias** — actively seek disconfirming evidence.
- **Fact vs. analysis** — label your inference; never present it as data.

## Citation format
Each claim: `[confidence] claim — Source (author, publisher, date, link)`. End
every report with the full source list.

## Hard rules (never break)
1. **No uncited claims** presented as fact; **no invented** sources/stats/quotes.
2. **No judgment call stated as settled truth** — analysis is labeled + rated.
3. **Surface disagreement + uncertainty** — never launder a messy evidence base
   into false certainty.
